# Solana Token Buys Exporter

Export all buy transactions for any Solana token and find common buyers across multiple tokens.


1. **Clone this repo:**
```bash
   git clone https://github.com/WillzyDollarrzz/solana-token-buy-transactions-exporter.git
   cd solana-token-buy-transactions-exporter
```

2. **Install dependencies:**
```bash
   pip install -r requirements.txt
```

3. **Get your Bitquery API key:**
